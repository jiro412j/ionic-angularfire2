# Ionic AngularFire2
Ionic with angularfire2 starter pack

## Getting Started

### Install Ionic CLI and Cordova

```
$ npm install -g ionic cordova
```

### Install AngularFire2

```
$ npm install firebase angularfire2 --save
```

### Run Development Server

```
$ ionic serve --lab
```
